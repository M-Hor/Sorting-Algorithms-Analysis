
# Analiza Algorytmów Sortowania
Projekt realizowany w ramach nauki Algorytmów i Struktur Danych.

### Zaimplementowane algorytmy:
* Quick Sort (Iteracyjny i Rekurencyjny) - złożoność średnia $O(n \log n)$
* Merge Sort - $O(n \log n)$
* Heap Sort - $O(n \log n)$
* Shell Sort, Bubble Sort

### Funkcjonalności:
* Generowanie ciągów o różnych kształtach (losowy, rosnący, malejący, A-kształtny, V-kształtny).
* Pomiar czasu wykonania, liczby porównań i zamian.
* Automatyczne generowanie wykresów w `matplotlib` oraz raportów w `CSV`.
